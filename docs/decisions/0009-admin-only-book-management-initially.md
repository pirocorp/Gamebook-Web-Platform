# ADR-0009: Admin-only book management initially

## Status

Accepted

## Context

The platform may eventually become a community platform, but the first version should be controlled and manageable.

## Decision

Only admins can add and manage gamebooks in the initial version.

## Reasons

- Reduces moderation needs.
- Reduces abuse risk.
- Keeps the first implementation focused.
- Makes content quality easier to control.
- Simplifies permissions.

## Consequences

Positive:

- Simpler MVP.
- Safer public launch.
- Easier validation workflow.

Negative:

- No community author uploads initially.
- Future contributor workflows will require additional roles and moderation tools.
