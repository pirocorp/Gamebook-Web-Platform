# Glossary

## Gamebook

An interactive book made of episodes and choices.

## Episode

A passage or numbered section in a gamebook.

## Choice

A possible continuation from one episode to another.

## Condition

A rule that determines whether a choice is available.

## Effect

A state change applied when a choice is selected.

## Player state

The digital adventure diary containing money, items, skills, code words, and notes.

## Aggregate root

A DDD entity that defines a consistency boundary and repository boundary.
